# MovieMatch

## Student Name
Alex Godino

## App Description
MovieMatch is a Streamlit app designed to help users discover movies based on their current mood. Instead of relying solely on genre or popularity, the app will use mood-based inputs (such as sad, nostalgic, or intense) to generate personalized movie recommendations. The goal is to create a more intuitive and user-centered way to choose what to watch.

## Intended Users
This app is intended for movie viewers who want a more personalized and efficient way to select films, especially those who prefer choosing based on emotional experience rather than traditional categories.

## Planned Features
- Mood-based movie selection interface
- Movie recommendations based on user-selected mood
- Brief explanations for each movie match
